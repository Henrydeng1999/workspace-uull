### 典型句式结构

#### 摘要开头

- "In this communication, we demonstrate a novel ..."
- "Herein, we report a facile procedure for ..."
- "Herein, we present a unique ... strategy ..."
- "Herein, we demonstrate for the first time that ..."

#### 背景引入

- "... has emerged as the inevitable route to achieve ..."
- "... has attracted extensive attention owing to ..."
- "Recently, ... has been emerging, which mainly relies on ..."
- "Since the pioneering work on ... in 1972, ... has attracted extensive attention owing to ..."

#### 转折对比

- "Unfortunately, ... bears an intrinsic drawback: ..."
- "Nevertheless, ... imply the success in ..."
- "In contrast to ..., ... shows ..."
- "However, ... remains poorly understood / limited / challenging."

#### 目的表达

- "In order to suppress ..., a widely used approach is to ..."
- "To this end, various efforts have to be made to ..."
- "To gain in-depth insight into ..., we compared ..."

#### 方法描述

- "... was synthesized/prepared/fabricated via/by a ... method"
- "... measurements were performed using ..."
- "The ... spectra were recorded on a ... instrument"

#### 结果展示

- "As shown in Figure X, ... exhibits/displays/shows/reveals ..."
- "These observations indicate that ..."
- "These data clearly demonstrate that ..."

#### 机理解释

- "This can be attributed to ..."
- "This is due to the strong electronic coupling at ..."
- "... arises from ..."
- "Such unique metallicity is ascribed to ..."

#### 强调意义

- "... plays a crucial/significant/key/pivotal role in ..."
- "... plays a dominant role in ..."
- "It is of great importance to ..."

#### 被动语态

- "... has been widely employed as ..."
- "... was measured/performed/recorded ..."
- "... can be assigned to ..."

#### 比较结构

- "... as compared to ..."
- "... in comparison with ..."
- "... is higher/lower than ..."

#### 因果关系

- "Consequently, ..."
- "Thus, ..."
- "As a result, ..."

#### 可能推测

- "It is expected that ..."
- "... may provide insight into ..."
- "... suggest that ... could originate from ..."

#### 总结展望

- "In summary, ..."
- "Taken together, ..."
- "This work opens up new possibilities for ..."
- "These findings pave the way for ..."

#### 定量描述

- "... is enhanced/improved by approximately / roughly / about X-fold / X% relative to ..."
- "... achieved drastically increased ... up to X times with respect to ..."

### 论文章节模板

#### Abstract

```
[背景] Recently, ... has attracted considerable attention as a promising ... for ... applications.
[问题] However, ... remains poorly understood / limited / challenging, hindering further development.
[方法] Herein, we report/present/demonstrate ... by using / via / through ...
[发现] The results show/indicate/reveal that ... exhibits/enables/demonstrates ...
[意义] This work provides deep insights into ... / offers a new strategy for ... / paves the way for ...
```

#### Introduction 段落组织

```
Para 1 [大背景]: ... has emerged as a promising candidate for ... owing to its unique ...
Para 2 [研究现状]: Numerous studies have focused on ... / Considerable efforts have been devoted to ...
Para 3 [存在问题]: Despite these advances, ... remains unclear / poorly understood / challenging.
Para 4 [本文工作]: Herein, we present ... / To address this issue, we design / In this work, we aim to ...
```

#### Methods

```
材料合成: ... was synthesized/prepared/fabricated via/by a ... method / according to a previously reported procedure with minor modifications.
表征手段: ... was characterized by ... / ... measurements were performed using ... / The ... spectra were recorded on a ... instrument.
测试条件: The ... experiments were conducted under ... conditions / at room temperature / in a ... atmosphere.
```

#### Results & Discussion

```
描述现象: As shown in Figure X, ... exhibits/displays/shows/reveals a pronounced/significant ...
解释机理: The ... can be attributed to / is ascribed to / arises from ... / This behavior originates from ...
对比讨论: Compared with ..., ... demonstrates/enables ... / In contrast to ..., ... shows ...
强调意义: These results suggest/indicate that ... plays a crucial/significant/key/pivotal role in ...
定量描述: The ... is enhanced/improved by approximately / roughly / about X-fold / X% relative to ...
```

#### Conclusion

```
总结: In summary/conclusion, we have demonstrated that ... / Taken together, our results reveal that ...
展望: This work opens up new possibilities for ... / Future work will focus on ... / These findings pave the way for ...
```

### 常见错误 & 修正

| 错误表达                                | 修正后                                                          | 说明                 |
| ----------------------------------- | ------------------------------------------------------------ | ------------------ |
| "The results are shown in Figure 1" | "As shown in Figure 1, ... exhibits ..."                     | 主动描述，避免被动堆砌        |
| "We did the experiment"             | "The experiments were conducted/performed"                   | 方法部分用被动语态          |
| "Very good performance"             | "Remarkably enhanced performance" / "Pronounced improvement" | 用学术评价词替代口语化表达      |
| "Because ... so ..."                | "Because ... , ..." / "Due to ..., ..."                      | 避免 because 和 so 连用 |
| "In this paper, we ..."             | "Herein, we ..." / "In this work, we ..."                    | 更正式的学术表达           |
| "make better"                       | "enhance" / "improve" / "boost"                              | 用精准动词替代模糊表达        |
| "look at"                           | "investigate" / "examine" / "explore"                        | 用学术动词替代口语          |
| "get"                               | "obtain" / "achieve" / "acquire"                             | 根据语境选择正式动词         |
